What You’re Seeing

This is the actual VP-8 depth data extracted from the Shroud of Turin — the same data used to generate all the 3D surfaces and AI reconstructions on this site. Each pixel’s height represents the image brightness at that point, which encodes the cloth-to-body distance. Brighter areas (nose, brow) were closest to the body; darker areas (background, eye sockets) were farthest away.

The Original depth map preserves all asymmetry from the source image. The Healed version has been bilaterally symmetrized to remove injury-related asymmetry, revealing the underlying facial geometry.

Try the Depth Heatmap material to see the elevation gradient, or Wireframe to see the mesh structure. Increase Z Exaggeration to amplify the facial relief.
